flow·bar

See what's in flight and switch tasks — right from your macOS menubar.

Download View on GitHub
brew tap pa/flow-bar https://github.com/pa/flow-bar brew install --cask flow-bar

What it does

Quick switcher

Click the “w”, type to filter your in-progress tasks, press Enter to jump in.

Overview dashboard

Exact metrics — in-progress, backlog, done, overdue, stale, owners, runs, projects. Tiles route you there.

Needs you

Owner questions, overdue, and waiting tasks gathered into one attention list.

Projects & playbooks

Drill into a project's tasks; see playbook runs and trigger new ones.

Owners

Status, next tick, parked questions — pause or resume in a click.

Flow roots

Switch between multiple FLOW_ROOTs — personal, work, a demo — from the footer.

flow-bar is a companion to the flow CLI — it reads your tasks via flow … --format json and switches with flow do. Requires macOS 13+ and flow on your PATH.
